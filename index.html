Introducing sikr.io

sikr.io is a password storage service, focused on keeping your most secure data really secure, that includes really sensitive data like SSL and SSH keys, wich you should keep always safe and at hand. If we have private data is for a reason, and we don't want anyone peeking on it, even under more than plausible excuses.

Secure

Such a big word nowadays, sikr is as secure as we could make it, you don't have username or password to login, instead we rely on a two phased authentication with you social network of choice, and since we don't trust them either, we generate a second code that will be stored on your browser, so the server will never contain data about you that can compromise anything.

Open Source

You should probably ask yourself, why should I trust you? The answer is simple: don't. If you're asking that question you probably want your own password manager, or to check what do we do to protect your data. Here you go, our platform is free and open source, because why not, and if you feel paranoid, just clone the code and deploy it on you own server!

When?

We expect to have a fist functional alpha really soon, and to release the first public version around mid or end 2015, now that's good news! Also, if you have spare time and you are interested, you can contribute with the project!.
